# Techifort Website

Modern software development & digital agency website built with Next.js 14, TypeScript, Tailwind CSS, Framer Motion, and next-themes.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Swap Assets Later

- Hero video: `public/videos/hero-space.mp4` (+ poster at `public/images/hero-poster.jpg`)
- Images: `public/images/`
- Team photos, portfolio, blog covers — replace placeholders in section data files under `lib/`
